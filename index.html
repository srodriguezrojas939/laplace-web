Guía de Uso

1. Variables s y t

Para la Transformada Directa, usa la variable t. Para la Inversa, usa la variable s.

Directa: sin(2*t)
Inversa: 1/(s^2 + 4)

2. Funciones Comunes

Usa exp(a*t) para $e^{at}$ y asegúrate de usar paréntesis en funciones trigonométricas.

Correcto: cos(3*t), exp(-2*t), t^3

3. Multiplicaciones

Usa siempre el asterisco * para indicar multiplicación, especialmente entre constantes y variables.

Ejemplo: 5*exp(-t) en lugar de 5exp(-t)
